Scan for unused media files and broken links with deletion and recovery options.

== Description ==

ELVO Media & Link Cleaner helps you maintain a clean and optimized WordPress site by:

* Scanning for unused media files that aren't referenced in your content
* Identifying broken external links throughout your site
* Providing bulk deletion options for unused media
* Offering configurable settings for scanning behavior

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/elvo-media-link-cleaner` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Navigate to ELVO Cleaner under the WordPress admin menu to start using the plugin

== Frequently Asked Questions ==

= What permissions do I need to use this plugin? =

You need to have administrator privileges to access the plugin settings and perform scans.

= Will this plugin delete my media files without confirmation? =

No, all deletion operations require explicit confirmation before proceeding.

= How does the plugin determine if a media file is unused? =

The plugin checks for:
- References in post content
- Usage as featured images
- References in custom fields
- Parent post relationships
